Se conocen cuatro tipos de algoritmos de machine learning. El primero es aprendizaje supervisado. Este algoritmo consiste en una variable dependiente que se debe predecir a partir de un conjunto de datos (variables independientes). A partir de las variables independientes se genera una función objetivo que busca hallar los valores óptimos que permitan tener el mejor modelo de predicción respecto a los valores originales. Ejemplos de este algoritmo son Regresiones lineal y logística, Arboles de decisión, Ramdon Forest.  El segundo algoritmo corresponde a aprendizaje no supervisado; con este algoritmo no se tiene ninguna función objetivo o variable dependiente a predecir. En este algoritmo se agrupan diferentes datos de acuerdo a sus atributos, es muy usado en la segmentación de clientes. Este algoritmo tiene dos subdivisiones: Algoritmos basados en centroides, algoritmos basados en densidad. El algoritmo basado en centroides calcula el punto medio entre una distribución de puntos, ejemplo k means. Los algoritmos basados en densidad buscan agrupar los puntos por cercanía, ejemplo K vecinos. El tercer algoritmo de aprendizaje es aprendizaje por refuerzo. Este algoritmo permite tomar decisiones específicas. El algoritmo se entrena continuamente aprendiendo a prueba y error, de tal manera que aprende de experiencias pasadas para tomar decisiones futuras, en cierta medida podría ser una combinación entre algoritmos supervisados y no supervisados.  Ejemplos de este algoritmo es procesos de decisión de Markov. El cuarto algoritmo es lenguaje semisupervisado el cual utiliza tanto datos etiquetados como no etiquetado. En todas las ocasiones se debe elegir entre algunos de los algoritmos ya vistos, para esta elección se debe tener en cuenta que se quiere conseguir y de que datos se disponen. Así como las siguientes consideraciones: Precisión, tiempo de entrenamiento, linealidad y cantidad de parámetros. La elección o evaluación de las consideraciones son específicas y discrecionales para cada caso.
